# Chrome Web Store 上架填写内容

## 基础信息

- 扩展名称：`土狗气象台`
- 简短描述：`在 gmgn、axiom、debot 页面上悬浮显示土狗气象台实时监控消息，支持风向精选、关键词种子和声音提醒。`

## 详细描述

土狗气象台是一款浏览器悬浮监控扩展，用来在交易与观察页面旁实时显示热点监控消息。

扩展当前会在 gmgn.ai、axiom.trade、debot.ai 页面内注入一个可拖动悬浮窗，实时接入土狗气象台的公开消息流，并提供未读统计、关键词种子、来源筛选、风向精选和声音提醒功能。

适合用于：

- 边看链上页面边接收热点监控消息
- 快速识别风向精选与题材变化
- 用关键词种子追踪特定叙事
- 在不切换标签页的情况下持续接收站内监控流

核心功能：

- 可拖动悬浮窗与最小化圆形浮窗
- WebSocket 实时消息
- 风向精选筛选
- 关键词种子
- 声音提醒
- 复制与打开原文链接
- 未读统计与全部已读
- 来源开关与分色区分

支持站点：

- gmgn.ai
- axiom.trade
- debot.ai

数据来源：

- https://tugoumeme.fun/

## 官方网站

如果 Chrome Web Store 后台的“官方网站”下拉仍然是“无”，说明你还没有在 Google Search Console 验证 `tugoumeme.fun`。

这项不是必须立即填，当前可以：

- 官方网站：先保持 `无`
- 首页网址：填 `https://tugoumeme.fun/`

如果后面你把 `tugoumeme.fun` 接到 Search Console 并验证所有权，再回来把官方网站下拉切成该站点即可。
